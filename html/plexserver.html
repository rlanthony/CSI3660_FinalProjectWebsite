How to Set Up Your Own Plex Server

Step 1:

Update your VM to ensure it has the latest configurations.

sudo apt-get update
sudo apt-get upgrade

Step 2:

Use the "curl" command to get the download link from the Plex website into your VM. This will add the Plex package to your VM along with the key. Once the key is added, the Plex repository can be added to the VM's sources list. Finally, refresh the packages list by updating the VM.

sudo curl https://downloads.plex.tv/plex-keys/PlexSign.key | sudo apt-key add -
sudo echo deb https://downloads.plex.tv/repo/deb public main | sudo tee /etc/apt/sources.list.d/plexmediaserver.list
sudo apt-get update

Step 3:

Now Plex can be installed in your server since the repositories have been added. Also, enable the Plex server to begin running when the VM is turned on.

sudo apt install plexmediaserver
sudo systemctl enable plexmediaserver
sudo systemctl start plexmediaserver

If you want to confirm the Plex Server is running, run the following command.

sudo systemctl status plexmediaserver.service

NOTE:

Several things have been done and added on to the VM by installing the Plex Server.

A "plex" user and group has been created for Plex to run under, in addition to a Plex Media Server directory that contains directories to monitor the server and the configuration information.

Step 4:

To access the Plex server on the web, enter the external IP address of your VM, followed by the 32400 port. You will be taken to a sign-in page that will prompt you to sign up to be able to access the Plex content.

http://SERVER_IP_ADDRESS:32400

Step 5:

Now you can upload files to the Plex server to watch on the web interface or mobile app. The file's location will be

/opt/plexmediaserver/movies or /opt/plexmediaserver/series
